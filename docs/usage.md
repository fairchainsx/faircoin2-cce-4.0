# Usage instructions

~~~
cd ~/faircoin2-cce-4.0

#### Start faircoin daemon & dbloader (by blocknotify) ####
./start_faircoind.sh

#### Stop faircoin daemon ############################
./stop_faircoind.sh

#### Start server ####################################
./start_server.sh

#### Stop server #####################################
./stop_server.sh

#### Run dbloader manually ###########################
./run_dbloader.sh
~~~
